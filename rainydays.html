RAINYDAYS

Online store for Rainydays made as a school project. Here you can find jackets made for active outdoor enthusiasts. The site should inspire them to push their comfort zone as well as make it easy for the user to find and buy jackets that suit their use.

Her har jeg holdt designet rent og ryddig, og bruker bilder for å engasjere og inspirere brukeren. Det presenteres kategorier på forsiden og brukeren kan klikke seg gjennom hele kjøpsprosessen. Man kan også lese mer om butikken, se handlekurv eller ta kontakt dersom det er nødvendig.

POTENSIALE:

  • Validere kontaktskjema, personopplysninger og påmeldig til nyhetsbrev med javascript.
  • Lagt inn produkter i API og dermed kunne vist faktiske produkter med produkt-spesifikke sider til hver av dem.
  • Laget karusell med nyheter på forsiden.
  • Laget fungerende funksjoner for å velge farge og størrelse på produkt.
  • Fungerende søkefelt
  • Laget hamburgermeny med Javascript

TECHNOLOGY:

HTML5, CSS, Figma, Netlify, GitHub
